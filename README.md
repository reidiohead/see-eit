# 📦 SEE-EIT
### Smart Equipment & Electronics Inventory Tracker

> A web-based laboratory equipment management system designed to streamline the borrowing, returning, and monitoring of laboratory equipment for the **Department of Industrial Engineering and Technology (DIET)** under the **College of Engineering and Information Technology (CEIT)** at **Cavite State University – Main Campus**.

---

## 📖 About

SEE-EIT is a laboratory equipment management system developed to modernize the traditional process of handling laboratory assets. It provides a centralized platform for managing equipment inventories, borrower records, borrowing requests, returns, and transaction history.

The system minimizes manual paperwork, improves accountability, and enables faculty and laboratory personnel to efficiently monitor the availability and condition of laboratory equipment.

---

## ✨ Features

- 🔐 Secure user authentication
- 👥 User and role management
- 📦 Equipment inventory management
- 📋 Borrowing request processing
- ↩️ Equipment return management
- 📅 Due date monitoring
- 🛠️ Equipment condition tracking
- 📊 Transaction history and logs
- 🔍 Search and filtering
- 📈 Dashboard with equipment statistics

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/see-eit.git
```

2. Navigate to the project

```bash
cd see-eit
```

3. Install dependencies

```bash
composer install
```

4. Copy the environment file

```bash
cp .env.example .env
```

5. Generate the application key

```bash
php artisan key:generate
```

6. Configure your MySQL database in `.env`

7. Run migrations

```bash
php artisan migrate
```

8. Start the development server

```bash
php artisan serve
```
